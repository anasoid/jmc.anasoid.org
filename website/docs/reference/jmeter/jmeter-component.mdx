# Jmeter component

Jmc Classes use the some name of Jmeter class with 'wrapper suffix'.

## Thread Group

All Thread group implements interface **ThreadWrapper**.

| Component                 | Covered | Class                   | Description      |
| ------------------------- | ------- | ----------------------- | ---------------- |
| **Thread Group**          | ✅      | ThreadGroupWrapper      | **Full support** |
| **setUp Thread Group**    | ✅      | SetupThreadGroupWrapper | **Full support** |
| **tearDown Thread Group** | ✅      | PostThreadGroupWrapper  | **Full support** |
