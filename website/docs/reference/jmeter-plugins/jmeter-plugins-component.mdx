# Jmeter plugins component

Jmc add also component from [jmeter-plugins](https://jmeter-plugins.org).

To use jmc plugins add dependency to jmc-jmeter-plugins see [dependencies](/docs/users/intro#dependencies).

## Custom Thread Groups

Artifact: [kg.apc:jmeter-plugins-casutg:**2.9**](https://jmeter-plugins.org/?search=jpgc-casutg)

| Component                           | Covered | Class                              | Description      |
| ----------------------------------- | ------- | ---------------------------------- | ---------------- |
| **Stepping Thread Group**           | ❌      | -                                  | **Deprecated**   |
| **Ultimate Thread Group**           | ✅      | UltimateThreadGroupWrapper         | **Full support** |
| **Concurrency Thread Group**        | ✅      | ConcurrencyThreadGroupWrapper      | **Full support** |
| **Arrivals Thread Group**           | ✅      | ArrivalsThreadGroupWrapper         | **Full support** |
| **Free-Form Arrivals Thread Group** | ✅      | FreeFormArrivalsThreadGroupWrapper | **Full support** |
